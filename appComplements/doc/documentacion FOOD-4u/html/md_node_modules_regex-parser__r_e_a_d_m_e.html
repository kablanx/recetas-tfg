regex-parser

A module that parses a string as regular expression and returns the parsed value.

:cloud: Installation

# Using npm
npm install --save regex-parser
# Using yarn
yarn add regex-parser

:clipboard: Example

// Dependencies
var RegexParser = require("regex-parser");
console.log(RegexParser("/^hi$/g"));
// => /^hi$/g
