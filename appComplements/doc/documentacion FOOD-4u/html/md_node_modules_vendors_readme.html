vendors

List of (real†) vendor prefixes known to the web platform. From Wikipedia and the CSS 2.1 spec.

† — real, as in, mso- and prince- are not included because they are not valid.

Install

npm:

npm install vendors

Use

var vendors = require('vendors')
console.log(vendors)

Yields:

[ 'ah',
'apple',
'atsc',
'epub',
'hp',
'khtml',
'moz',
'ms',
'o',
'rim',
'ro',
'tc',
'wap',
'webkit',
'xv' ]
