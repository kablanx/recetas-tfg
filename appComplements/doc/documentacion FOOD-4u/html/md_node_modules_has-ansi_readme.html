has-ansi

Check if a string has ANSI escape codes

Install

$ npm install --save has-ansi

Usage

var hasAnsi = require('has-ansi');
hasAnsi('\u001b[4mcake\u001b[0m');
//=> true
hasAnsi('cake');
//=> false
